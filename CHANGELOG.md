# Changelog

## [v0.5.0] - 2020-03-04

### Dependencies

- Updated `cose` dependency version constraint from `0.11.x` to `1.x`

## [v0.4.0] - 2020-01-30

### Dependencies

- Updated `cose` from `v0.9.0` to `v0.11.0`

## [v0.3.0] - 2019-09-02

### Added

- `CWT.decode` can decode a MACed CWT

## [v0.2.0] - 2019-08-31

### Added

- `CWT.decode` can decode a signed CWT
